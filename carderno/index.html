Caderno de Anotações

Titulos

Usa a tag < h1 > "..." < h1 > para fazer um titulo, respeitando a hierarquia.;

Parágrafos

Você pode escrever um paragrafo de qualquer jeito. Basta colocar tudo no meio de par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar espeífico como esse,
você pode usar a tag < br >.

Adicionar imagens

Para adicionar imagens basta usar a tag < img > e colocar o caminho do arquivo, desde que o mesmo estea na mesma pasta ou em uma subpasta

É possivel colocar uma logo de imagens exeternas

Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (não semantica)

Nesta frase, temos um termo em destaque usando a tag STRONG (semãntica)

Italico / Ênfase

Nesta frase, temos um termo em italico usando a tag I (nâo semântico)

Nesta frase, temos um termo em ênfase usando o tag EM (semãntica)

Texto marcado

Podemos criar tabém um texto marado usando a tag MARK

e no outro paragrafo temos outro texto marcado no final.

Texto grande e pequeno

Estamos cirando um texto grande e um texto pequeno.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântico)
